# WRC_UTILITY
Utility Components

## Download
Grab library via Gradle
```groovy
compile 'com.wildnet.wrcUtility:wrcUtility:1.0.9'
```
or Maven
```xml
<dependency>
  <groupId>com.wildnet.wrcUtility</groupId>
  <artifactId>wrcUtility</artifactId>
  <version>1.0.9</version>
  <type>pom</type>
</dependency>
```
